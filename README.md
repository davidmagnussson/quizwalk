# Description
A project in _DM2518 Mobile Development_

QuizWalk is a mobile application for both _Android_ and _iOS_. It uses the technology of beacons to display different question if you go near beacon.

```
public/
  app.js
  auth.js
  evothings.json
  index.html
  model.js
```
